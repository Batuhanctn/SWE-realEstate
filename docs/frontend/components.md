# Components

## Pages

### Home
The main landing page showing featured properties and search functionality.

### PropertyList
Displays a grid or list of properties with filtering and sorting options.

### PropertyDetails
Shows detailed information about a specific property.

### UserProfile
User profile management and settings.

### Favorites
Lists user's favorite properties.

## Common Components

### Header
- Navigation menu
- User authentication status
- Search bar

### PropertyCard
- Property image
- Basic property information
- Favorite button

### SearchFilters
- Price range
- Location
- Property type
- Other filters

### AuthForms
- Login form
- Registration form
- Password reset form
